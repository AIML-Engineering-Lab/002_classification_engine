\section{Project Overview}

This project creates a binary classification pipeline using logistic regression from scratch, incorporating the sigmoid function, cross-entropy loss, L1/L2 regularization, and handling class imbalance with SMOTE. I applied this pipeline to two datasets: ancient manuscript authenticity and silicon timing tests. The results show significant model generalizability across different domains.

\textbf{Sigmoid Function:} Maps log-odds to probabilities, creating a decision boundary.

\textbf{Cross-Entropy Loss:} Convex function that penalizes confident incorrect predictions.

\textbf{Gradient Descent:} Iterative optimization technique for tuning model parameters.

\textbf{L1 Regularization:} Penalizes coefficients to promote sparse models.

\textbf{L2 Regularization:} Shrinkage technique to control overfitting.

\textbf{SMOTE:} Generates synthetic samples for minority classes to combat imbalance.

\textbf{Threshold Tuning:} Optimizes classification thresholds based on cost ratios.

\textbf{AUC-ROC:} Measures model discrimination across thresholds.

\textbf{MCC:} Provides a balanced measure of classification performance.

\textbf{Platt Scaling:} Calibrates model probabilities using logistic regression.
