\section{Project Overview}

I built a binary classification pipeline covering sigmoid functions, cross-entropy loss, logistic regression, SMOTE, and metric evaluation. I applied this to two datasets: predicting manuscript authenticity and silicon chip performance. The results showed strong generalization across both domains.

\textbf{Sigmoid Function:} Maps log-odds to probabilities, defining the decision boundary.

\textbf{Cross-Entropy Loss:} Convex loss function that penalizes confident wrong predictions.

\textbf{Gradient Descent:} Optimization method for parameter fitting in logistic regression.

\textbf{SMOTE:} Technique to address class imbalance by generating synthetic samples.
